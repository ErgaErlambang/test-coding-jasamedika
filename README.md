# Sewa Mobil ( Flash RentCar )


## Installation

```bash
# install package
$ composer install

# Copy Env
$ cp .env.example .env

# generate key laravel
$ php artisan key:generate

# migrate db + seeder for dummy data
$ php artisan migrate --seed

```
